## 人物

男子：普通古代人，背着箩筐，没有拿拐杖。![参考图](./%E5%8F%A4%E4%BB%A3%E7%94%B7%E5%AD%90.jpeg)

年龄：26
身高：165
性别：女
性格：内向，内心只对喜欢的人敞开，想很多东西，一句话就能让她内心有起伏，有小动作，小心思

## 人物关系

女子是男子曾经的恋人，而男子忘了女子，所以女子会各种献媚，而男子却很绅士。
